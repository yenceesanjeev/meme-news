Meme News API

This is the API endpoint for Meme News. Access the data at:

./memes.json

JSON Structure

{
  "news": [
    {
      "title": "News Title",
      "date": "2024-12-16 04:32:00",
      "generated_at": "2024-12-16 22:19:14",
      "meme_files": [
        "meme_20241216_221914_1.jpg",
        "meme_20241216_221914_2.jpg",
        "meme_20241216_221914_3.jpg",
        "meme_20241216_221914_4.jpg"
      ]
    }
  ]
}

Images

Access meme images at:

./meme_images/[filename]
